# to n from mk — letter-writing app

a letter-writing experience.
choose the type.
choose the tone.
answer what you can.

this is not a notes app.
this is not a journal.
this is a moment.

## concept
to n from mk turns structured answers into letters written with my words.
cryptic and intentional.

## what i worked on
- designed the ui (graispect / mk’s cams visual language: dark, minimal, glitch-leaning).
- built the letter flow: type → tone → prompts → generation.
- built the prompt system + question bank by structuring categories & pacing.
- implemented generation through a secure backend (api key).
- added constraints (length rules + output shaping).

## features
- type + tone based letter generation
- structured question flow (identity / personal / analytical / emotional / casual)
- custom “mk voice” shaping (lexicon + style rules)
- character constraints (to fit my formated rules)
- save / copy / export
- privacy-first: no social graph, no feed

## status
in active development.
the repository is public for portfolio visibility, but full implementation details are intentionally limited.

## stack
- Swift / SwiftUI
- Cloudflare Worker (API proxy)
- OpenAI API
- JSON lexicon + prompt rules
- HTML / CSS (policy + support pages)
